# Brand Specification: Modern Minimal Blog

## Visual Direction: Modern Minimal
Inspired by Linear, Vercel, and Stripe. Precise, software-native, and crisp.

## Color Tokens (OKLch)
- `--bg`:      `oklch(99% 0.002 240)` (Off-white canvas)
- `--surface`: `oklch(100% 0 0)` (Pure white cards/panels)
- `--fg`:      `oklch(18% 0.012 250)` (Deep grey-blue text)
- `--muted`:   `oklch(54% 0.012 250)` (Muted secondary text)
- `--border`:  `oklch(92% 0.005 250)` (Subtle hairlines)
- `--accent`:  `oklch(58% 0.18 255)` (Crisp blue primary)
- `--error`:   `oklch(62.8% 0.25 29.2)` (Status red)
- `--success`: `oklch(70% 0.18 144)` (Status green)

## Typography
- **Display Font**: `-apple-system, BlinkMacSystemFont, 'SF Pro Display', system-ui, sans-serif`
- **Body Font**: `-apple-system, BlinkMacSystemFont, 'SF Pro Text', system-ui, sans-serif`
- **Mono Font**: `'JetBrains Mono', 'IBM Plex Mono', ui-monospace, monospace`

## Layout Posture
- **Radii**: `6px` for a precise, sharp look.
- **Borders**: `1px` hairline borders.
- **Shadows**: Minimal elevation. Only for modals/dropdowns.
- **Spacing**: Generous whitespace to allow content to breathe.
- **Letter Spacing**: `-0.02em` for large display headlines.
